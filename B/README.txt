L'applicazione è possibile eseguirla in ambiente windows mediante i file .jar del server e del client. 

Come prima cosa andrebbe avviato il proprio server postgresql sul proprio pc, in seguito impostata come password per l'utente "postgres" proprio "postgres"
attraverso la shell di postgresql. (Qui il link per tale modifica https://commandprompt.com/education/how-to-reset-forgotten-password-for-postgres-user/)

Una volta impostata la password per il relativo utente admin, si dovrà avviare prima l'eseguibile del server e inserire le credenziali del database(es. localhost, porta 5432, postgres, postgres), così da avere il servizio attivo e poi quella del client così da potersi connettere al medesimo.
